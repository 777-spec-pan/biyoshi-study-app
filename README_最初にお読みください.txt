美容師国家試験 MASTER V2.1

【追加機能】
・お気に入り
・問題検索
・模擬試験100分タイマー
・結果画面の分野別成績
・収録問題数表示
・更新確認ボタン
・古いキャッシュを削除する仕組み
・学習データのリセット

【アップロード方法】
1. ZIPを解凍
2. 中の全ファイルとdataフォルダを選択
3. GitHubのbiyoshi-study-appへ上書きアップロード
4. Commit changes
5. 公開画面で ?v=21 を付けて開く
   例：https://777-spec-pan.github.io/biyoshi-study-app/?v=21

【重要】
dataフォルダも必ずアップロードしてください。
